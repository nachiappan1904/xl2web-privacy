Privacy Policy for XL2Web

Effective Date: [19-02-2026]

XL2Web is designed to help users automatically fill web forms using data they provide locally.

Data Collection

XL2Web does not collect, transmit, store on external servers, or sell any personal data.

Local Processing

All Excel/CSV files, field mappings, and settings are stored locally in the browser using Chrome storage and IndexedDB.

Website Access

XL2Web requires access to websites to detect and fill form fields chosen by the user, including forms inside embedded iframes. This access is used only for autofill functionality.

Third-Party Sharing

We do not share or sell user data to any third parties.
