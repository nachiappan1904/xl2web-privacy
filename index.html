Privacy Policy for XL2Web

Effective Date: [fill]

XL2Web is designed to help users automatically fill web forms using data they provide locally.

Data Collection

XL2Web does not collect, transmit, store on external servers, or sell any personal data.

Local Processing

  • Reads Excel/CSV files selected by the user
  • Stores configuration locally using Chrome storage and IndexedDB
  • Fills forms only when triggered by the user

Website Access

XL2Web requires access to websites in order to detect and fill form fields chosen by the user, including forms inside embedded iframes. This access is used solely for form autofill functionality.
